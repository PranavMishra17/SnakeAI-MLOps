// Load trained PyTorch models in C++ using LibTorch auto dqnAgent = std::make_unique<DQNAgent>(modelInfo); Direction action = dqnAgent->getAction(gameState, false);

Models are trained in Python using PyTorch, then exported and loaded in C++ using LibTorch for real-time inference during gameplay. This hybrid approach combines Python's flexibility for experimentation with C++'s performance for deployment.
